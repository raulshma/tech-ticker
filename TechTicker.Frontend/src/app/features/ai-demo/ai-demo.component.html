help_outline How to Use the AI Assistant Component Learn how to integrate and customize the AI assistant in your own components integration_instructions Basic Integration How to add the AI assistant to your components

Import the AI assistant component and add it to your template with a configuration object: 

<app-ai-assistant 
  [config]="myConfig"
  (result)="onResult($event)"
  (loading)="onLoading($event)">
</app-ai-assistant>
 settings Configuration Options Available configuration properties and their usage

title

Display title for the assistant

subtitle

Subtitle description text

placeholder

Input field placeholder text

defaultSystemPrompt

Default system prompt for AI behavior

defaultJsonSchema

Default JSON schema for structured output

showAdvancedOptions

Show/hide advanced configuration panel

 code Example Implementation Complete example with TypeScript configuration 
export class MyComponent {
  myConfig: AiAssistantConfig = {
    title: 'Custom AI Assistant',
    subtitle: 'Specialized for my use case',
    placeholder: 'Enter your specific request...',
    defaultSystemPrompt: 'You are a helpful assistant specialized in...',
    showAdvancedOptions: true,
    allowJsonSchema: true
  };

  onResult(result: AiAssistantResult) {
    console.log('AI Response:', result);
    // Handle the AI response
  }

  onLoading(isLoading: boolean) {
    // Handle loading state
  }
}
